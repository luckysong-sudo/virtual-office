# 📊 数据与产品技能包

## 可用技能

### 1. 数据分析
- **描述**: 分析业务数据
- **能力**:
  - 用户行为分析
  - 性能指标统计
  - A/B测试结果
  - 趋势预测

### 2. 产品规划
- **描述**: 制定产品路线图
- **能力**:
  - 需求优先级排序
  - MVP定义
  - 功能拆解
  - 里程碑规划

### 3. 用户研究
- **描述**: 理解用户需求
- **能力**:
  - 用户画像构建
  - 用户体验地图
  - 痛点分析
  - 竞品调研

### 4. 报告生成
- **描述**: 自动生成各类报告
- **能力**:
  - 日报/周报
  - 性能报告
  - 质量报告
  - 项目进度报告

### 5. 通知与沟通
- **描述**: 团队沟通协调
- **能力**:
  - 发送通知
  - 创建会议纪要
  - 任务分配
  - 状态同步
