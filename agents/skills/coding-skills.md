# 🧩 编程技能包 (Coding Skills)

## 可用技能

### 1. 代码编辑 (File Edit)
- **描述**: 读取、编写、编辑代码文件
- **适用角色**: Bob(后端), Henry(前端), Frank(全栈)
- **能力**: 
  - 读取任意代码文件
  - 编写新功能模块
  - 重构现有代码
  - 修复Bug

### 2. 代码执行 (Exec)
- **描述**: 在沙箱中执行命令
- **适用角色**: Bob, Henry, David, Eve, Grace
- **能力**:
  - 运行测试脚本
  - 编译代码
  - 执行构建命令
  - 调试程序

### 3. Git操作 (GitHub)
- **描述**: 版本控制操作
- **适用角色**: 全员
- **能力**:
  - commit/push/pull
  - 创建分支
  - 提交PR
  - 管理Issues

### 4. 原型验证 (Spike)
- **描述**: 快速验证技术可行性
- **适用角色**: Bob, Henry, David, Frank
- **能力**:
  - 创建最小可行原型
  - 对比技术方案
  - 输出验证报告

### 5. 架构绘图 (Diagram Maker)
- **描述**: 生成架构图/流程图
- **适用角色**: Bob(Henry), Frank
- **能力**:
  - 生成SVG架构图
  - 生成时序图
  - 生成数据流图

### 6. 技能创建 (Skill Creator)
- **描述**: 创建新的可复用技能
- **适用角色**: Bob, Henry, Frank
- **能力**:
  - 定义技能描述
  - 编写技能脚本
  - 注册新技能

### 7. 任务编排 (TaskFlow)
- **描述**: 协调多步骤后台任务
- **适用角色**: Bob, Frank
- **能力**:
  - 创建持久化任务流
  - 子任务管理
  - 状态跟踪

### 8. 网络搜索 (Web Search)
- **描述**: 搜索最新技术和解决方案
- **适用角色**: 全员
- **能力**:
  - 搜索技术方案
  - 查阅文档
  - 获取最新资讯

### 9. 页面抓取 (Web Fetch)
- **描述**: 获取网页内容
- **适用角色**: 全员
- **能力**:
  - 抓取API文档
  - 获取教程
  - 分析竞品

### 10. 日志查看 (Process)
- **描述**: 查看和管理进程
- **适用角色**: Bob, David, Eve
- **能力**:
  - 查看进程日志
  - 监控运行状态
  - 管理后台任务
