# 🎨 设计技能包 (Design Skills)

## 可用技能

### 1. CSS设计与优化
- **描述**: 创建和维护设计系统
- **能力**:
  - 定义CSS变量系统
  - 创建响应式布局
  - 优化动画性能
  - 设计暗色模式

### 2. Canvas渲染
- **描述**: HTML5 Canvas图形绘制
- **能力**:
  - 绘制办公室平面图
  - 渲染角色动画
  - 实现粒子效果
  - 优化渲染性能

### 3. 用户体验设计
- **描述**: 优化用户交互流程
- **能力**:
  - 设计用户引导流程
  - 创建交互反馈
  - 优化表单体验
  - 设计错误提示

### 4. 图标与视觉
- **描述**: 创建视觉元素
- **能力**:
  - 设计SVG图标
  - 创建表情符号
  - 设计配色方案
  - 制作Meme图片

### 5. 无障碍设计
- **描述**: 确保所有人都能用
- **能力**:
  - 添加ARIA标签
  - 键盘导航支持
  - 屏幕阅读器兼容
  - 对比度检查
