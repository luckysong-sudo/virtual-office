# 🔧 DevOps技能包

## 可用技能

### 1. 服务器管理
- **描述**: 管理运行环境
- **能力**:
  - 启动/重启服务
  - 端口管理
  - 进程监控
  - 日志查看

### 2. 安全加固
- **描述**: 保护系统安全
- **能力**:
  - CORS配置
  - 输入验证
  - 敏感信息管理
  - HTTPS配置

### 3. 部署自动化
- **描述**: 自动化部署流程
- **能力**:
  - 自动构建
  - 自动测试
  - 自动发布
  - 回滚机制

### 4. 监控告警
- **描述**: 系统健康监控
- **能力**:
  - 服务状态检查
  - 性能指标监控
  - 异常告警
  - 日志聚合

### 5. 环境配置
- **描述**: 管理环境变量
- **能力**:
  - .env文件管理
  - 多环境配置
  - 密钥管理
  - 配置热更新
