## ✨ 核心功能

- **🏠 虚拟办公室** — 2D 网格地图，8 位角色自由走动、交互
- **🗣️ 对话系统** — 角色间可发起对话，支持多轮交流
- **📋 任务管理** — 每个角色有当前任务和进度追踪
- **😊 情绪系统** — 心情与精力影响角色行为和生产力
- **🎨 暗色/亮色模式** — 完整的主题切换支持
- **⌨️ 键盘快捷键** — 快速导航和操作
- **📊 实时状态 API** — 获取办公室和角色的实时数据
- **🔄 自优化系统** — 多轮迭代持续改进代码质量
