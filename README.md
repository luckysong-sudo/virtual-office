## ✨ 核心功能

- **🏢 虚拟办公室空间** — 交互式 2D 办公室地图，角色自由移动
- **👥 8 个独特角色** — 每位角色拥有独立的性格、情绪和生产力模型
- **💬 实时对话系统** — 角色间基于上下文的自然对话

- **🎨 响应式 UI** — 暗色模式支持，精美的 CSS 设计令牌
- **🛡️ 安全中间件** — API 端点白名单验证、请求速率限制、CORS 防护

- **⌨️ 键盘快捷键** — 全局导航和操作快捷方式
- **🔄 自优化系统** — AI Agent 自主发现瓶颈并提出改进方案
