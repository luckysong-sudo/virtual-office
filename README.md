## ✨ 核心功能

- **🏢 虚拟办公室** — 8 位 AI 角色在 2D 网格上自由移动、交谈、协作
- **🎭 个性系统** — 每位角色有独特的性格、情绪状态和精力水平
- **💬 实时聊天** — 角色间可以互相发送消息，模拟真实办公室沟通
- **📊 仪表盘** — 实时监控团队状态、生产力、情绪变化
- **⌨️ 键盘快捷键** — 快速导航和操作
- **🌙 暗色模式** — 支持明暗主题切换
- **🔔 通知系统** — 事件提醒和消息推送
- **📱 响应式设计** — 适配桌面和移动端
- **🔄 自优化系统** — 智能体自主分析、修复和改进代码
