## ✨ 核心功能

- **🏢 虚拟办公室** — 8 位 AI 代理在共享空间中有独立工位，可移动、交谈、协作
- **🧠 自主决策** — 每位代理有独特人格和决策逻辑，能主动选择任务
- **💬 自然对话** — 代理之间可以进行多轮对话，共享上下文
- **📋 任务系统** — 代理从任务池中选择工作，完成后自动获取新任务
- **🎭 动态情绪** — 心情和精力随时间变化，影响工作效率和行为
- **🔧 自优化引擎** — 定期分析代码库并自动改进，越用越好
- **🌙 日夜循环** — 办公室根据真实时间调整氛围和行为模式
- **📊 实时监控** — 通过 WebSocket 实时查看办公室状态

## 🤖 自优化系统 v5.0

Virtual Office 的自优化引擎是项目的核心创新。每一轮优化中：

1. **📋 知识收集** — 所有代理阅读整个代码库，建立共享知识
2. **🧠 独立分析** — 每位代理专注于自己负责的模块，识别改进点
3. **📝 提案生成** — 代理生成具体的代码修改建议（patches）
4. **🗳️ 投票决策** — 所有代理对提案进行投票，选出最优方案
5. **🔨 执行应用** — 通过的修改被应用到代码库
6. **✅ 验证测试** — 服务器重启并验证功能正常

经过 18 轮优化迭代，代码库已从最初的简单演示发展为功能完整的虚拟办公室系统，累计产生 **152 次代码变更**。
