## ✨ 核心功能

- **🏗️ 虚拟办公室空间** — 2D 网格地图，支持 8 个智能体自由移动与交互
- **👥 多角色 AI Agent** — 每个角色拥有独特性格、部门和行为模式
- **💬 实时对话系统** — 角色间基于距离和角色的自然对话触发
- **📊 动态状态追踪** — 心情、精力、生产力等指标实时变化
- **🎨 精美 UI 界面** — 响应式设计，暗色模式，动画效果
- **📡 RESTful API** — 完整的后端接口支持状态查询与交互
- **🤖 自优化系统** — Agent 知识库持续进化，每轮迭代自我改进

## 🤖 自优化系统 v5.0

Virtual Office 引入了创新的自优化机制，让每个 AI 角色都能从经验中学习：

- **🧠 知识积累** — 每个 Agent 维护独立的知识库，记录交互经验和最佳实践
- **🔄 迭代优化** — 每轮优化后 Agent 自动分析反馈并调整策略
- **📈 性能追踪** — 生产力、心情、精力等指标驱动的行为自适应
- **🎯 目标导向** — Agent 根据角色职责自主分配和优化任务
- **📋 Patch 系统** — 每次优化生成可追溯的变更补丁，记录修改内容
