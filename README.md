## ✨ 核心功能

- **🏢 虚拟办公室** — 8 名 AI 角色在 2D 办公室中自由走动、互动
- **💬 实时对话** — 点击角色触发对话气泡，了解他们的心情和任务
- **📊 状态面板** — 监控每位角色的心情、精力和生产力的实时仪表盘
- **🎮 交互控制** — 拖拽角色移动，双击打开详情面板
- **🌙 日夜循环** — 办公室环境随时间变化，影响角色行为
- **🔔 通知系统** — 角色间消息传递和事件提醒
- **📈 自优化系统** — 团队自动学习和改进工作流程
