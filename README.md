## ✨ 核心功能

- **🏢 沉浸式虚拟办公室** — 2D 俯视图展示角色在办公室中自由走动、交谈、协作
- **👥 多角色 AI 代理** — 8 位各具性格的虚拟员工，拥有独立情绪、精力与行为模式
- **💬 实时对话系统** — 角色间随机交流、会议讨论、任务协调
- **🎯 任务管理系统** — 角色根据优先级接受并完成各类办公任务
- **😊 情绪与精力模型** — 心情、精力、疲劳度动态变化，影响工作效率
- **🎨 自定义办公室布局** — 拖拽调整工位，自定义办公空间
- **📊 实时数据面板** — 查看团队状态、任务进度、情绪趋势
- **⌨️ 键盘快捷键** — 支持快速操作和导航
- **🌙 暗色模式** — 保护视线的深色主题
- **🔄 自优化系统** — AI 代理持续分析并改进自身代码

## 🤖 自优化系统 v5.0

Virtual Office 配备了独特的自优化引擎，让 AI 代理能够：

1. **🔍 知识检索** — 从知识库中查找相关修复方案和经验教训
2. **🧠 问题分析** — 识别代码中的潜在问题和改进机会
3. **📋 制定计划** — 生成具体的优化补丁列表
4. **🛠️ 应用修改** — 对 server.js、index.html、CSS 等文件实施改进
5. **📚 知识沉淀** — 将经验存入知识库供后续迭代使用

每轮优化后，系统会自动记录变更详情到 `.optimization_history/` 目录，形成完整的进化日志。
