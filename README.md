## ✨ 核心功能

- **🏢 虚拟办公室** — 3D 风格的开放办公空间，角色在工位间自由走动
- **📋 任务系统** — 每个角色有独立任务队列和进度追踪
- **💬 对话交互** — 点击角色查看状态、发送消息、了解当前工作
- **📊 实时仪表盘** — 全局统计面板展示团队健康度
- **🌙 日夜交替** — 办公室环境随时间自动切换明暗模式
- **⌨️ 键盘快捷键** — 支持 WASD 移动和 Tab 角色切换
- **🔔 通知系统** — 角色状态变化时推送通知提醒
- **📱 响应式设计** — 适配桌面端和移动端浏览器

## 🤖 自优化系统 v5.0

### 工作原理

1. **📚 知识库学习** — 每个角色拥有专属知识模块，记录代码规范和最佳实践
2. **🔍 问题检测** — 自动扫描代码库中的潜在问题和优化机会
3. **🛠️ 自主修复** — 角色根据知识库中的补丁方案自行修改代码
4. **📊 效果评估** — 每轮优化后评估变更效果，生成优化报告
