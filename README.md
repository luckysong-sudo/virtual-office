## ✨ 核心功能

- **🏠 虚拟办公室** — 直观的 2D 空间布局，角色自由移动和交互
- **💬 实时对话** — 团队成员之间模拟自然工作沟通
- **📋 任务系统** — 每人有独立当前任务，状态动态更新
- **🎭 个性系统** — 每个角色拥有独特性格和行为模式
- **📊 状态面板** — 心情、精力、生产力实时可视化
- **🌙 日夜循环** — 办公室状态随时间自动切换
- **⌨️ 键盘快捷键** — 快速导航和控制
