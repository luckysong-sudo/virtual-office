## ✨ 核心功能

- **🏢 虚拟办公室可视化** — 交互式 2D 办公室布局，角色在工位间自由移动
- **🗣️ 对话系统** — 角色之间可以交谈，通过气泡对话框展示互动
- **🤖 8 种独特角色** — 每个 AI 员工拥有独立的性格、工作习惯和行为模式
- **📊 实时仪表盘** — 监控所有员工的效率、心情和能量水平
- **🌙 日夜循环** — 模拟真实办公时间，影响员工行为和状态
- **⌨️ 键盘快捷键** — 快速导航和操作办公室功能
- **🎨 暗色/亮色主题** — 支持 CSS 设计令牌切换视觉风格
- **📱 响应式设计** — 适配桌面和移动端浏览器

## 🤖 自优化系统 v5.0

Virtual Office 内置了一个多代理自优化引擎：

1. **🧠 知识图谱** — 每个角色维护自己的专业知识库，包含角色洞察、同事画像和工作偏好
2. **🔧 自我修复** — 系统自动检测问题并生成修复补丁
3. **📚 经验学习** — 从历史优化中积累经验，不断提升协作效率
4. **🔄 迭代优化** — 每轮优化后，角色基于新知识进行下一轮改进
5. **📊 版本追踪** — 完整的变更历史和可追溯的优化路径

> 系统已经历 18 轮自主优化，从最初的简单演示演进为具有复杂交互能力的虚拟办公室。
