# 🏢 Virtual Office — 智能虚拟办公室

一个由 AI 代理驱动的虚拟办公室模拟系统，8 位性格迥异的数字员工在虚拟空间中协作、编码、设计、测试与监控。系统内置自优化引擎，持续迭代改进自身架构。

> **项目理念：** 让 AI 像人类一样办公——有情绪、有精力、有分工、有节奏。

## ✨ 核心功能

- **🏠 虚拟办公室场景** — 可交互的 2D 办公室空间，角色自由移动
- **👥 8 位 AI 代理** — 各自拥有独特性格、情绪系统和行为模式
- **💬 对话系统** — 代理之间可以交谈，支持语音合成 (TTS)
- **📋 任务系统** — 代理分配并执行独立任务
- **🎭 情绪模型** — 心情和精力影响代理行为和生产力
- **🌙 昼夜节律** — 白天高效工作，夜晚进入休息模式
- **⌨️ 键盘快捷键** — 快速切换视角、控制动画、打开菜单
- **🎴 CSS 暗色模式** — 一键切换主题，保护视力
- **📈 性能监控** — 实时追踪请求速率、错误率和资源使用

## 🤖 自优化系统 v5.0

Virtual Office 拥有独特的自优化引擎，每轮迭代自动执行以下流程：

1. **📝 知识沉淀** — 每个代理将经验写入共享知识库 (knowledge.json)
2. **🔍 问题发现** — 代理扫描代码和配置，识别潜在改进点
3. **🛠️ 补丁生成** — 基于知识积累生成具体修复/优化补丁
4. **📦 补丁应用** — 按优先级依次应用，确保系统稳定性
5. **🔄 版本升级** — 记录变更、递增版本号、提交 Git

目前已完成 **18 轮** 自优化迭代，累计变更覆盖后端、前端、样式和配置全栈。
