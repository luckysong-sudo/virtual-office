## ✨ 核心功能

- **🏗️ 3D 虚拟办公室** — 基于 Three.js 的沉浸式 3D 办公场景
- **🤖 AI Agent 团队** — 8 名具有独立人格和技能的虚拟员工
- **🎭 实时动画** — 角色移动、对话气泡、表情变化
- **💬 交互系统** — 点击角色查看详情，与团队互动
- **🌙 日夜循环** — 自动切换白天/夜间模式
- **⌨️ 键盘快捷键** — 支持键盘导航和操作
- **📱 响应式设计** — 桌面端和移动端均可访问
- **🔄 自优化系统** — 团队通过迭代轮次自主改进代码
