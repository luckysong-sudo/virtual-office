# 🏢 虚拟办公室技能注入系统

> 将 OpenClaw 的 50+ 技能注入到 8 个 AI 角色的能力库中
> 每个角色获得与其岗位相关的技能工具集
> 角色可以随时自主调用这些技能来完成任务

## 注入方式

每个角色的 system_prompt 中添加了技能描述和调用指南
技能配置文件: agents/skills/*.md
运行时通过 API 模拟技能调用
